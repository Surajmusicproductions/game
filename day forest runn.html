CRITICAL ERROR: Google Maps API Key Missing

This application cannot function without a valid Google Maps API key.

To fix this:

  1. Open the synth.html file in an editor.
  2. Find the script block at the bottom of the <body>.
  3. Replace 'YOUR_API_KEY_HERE' with your actual key from the Google Cloud Console.

Ensure the "Maps JavaScript API" is enabled for your key.
